# Cling - Press Kit

## About

Cling is an instant fuzzy file finder for macOS. Find any file in under 100ms and act on it in the same instant. Press Right Command + / (slash) to show or hide it.

Cling keeps a custom binary index of your whole filesystem, including system files, hidden files, dotfiles and app data that Spotlight ignores. It's "Everything for Mac": keyboard-centric, minimalistic, and built for power users.

## Features

- Instant fuzzy search across millions of files in under 100ms
- Finds system files, hidden files, dotfiles and app data that Spotlight misses
- Keyboard-centric actions: Open with…, copy paths, batch rename, QuickLook, drag and drop
- Smart defaults show your most recently changed files on launch
- Search history with arrow-key navigation and tab completion
- Extension-aware queries like ".png icon" or ".pdf invoice"
- Quick Filters: Option + a single key to narrow by file type
- Folder and Volume filters saved as named hotkeys
- Custom scripts in any language, run on selected files with a hotkey
- External volume indexing for USB drives, external disks and network shares, persistent even while unmounted
- Exclude files with a ~/.fsignore file using gitignore patterns
- Raycast extension for searching without leaving Raycast

## Use Cases

- Finding a file by a rough name and opening or copying it instantly
- Locating dotfiles, app data and system files Spotlight won't show
- Sending a found file to a shelf like Yoink or uploading with Dropshare
- Running a custom script on selected files
- Searching external and network volumes, even when unmounted
- Narrowing to a file type with a single Quick Filter key

## How It Compares

Cling is similar to Spotlight, Alfred and Raycast for instant results, but adds fuzzy search, searches files those tools index, and narrows by extension. It is not like ProFind, HoudahSpot or EasyFind: it is for quickly finding one or more specific files by roughly knowing the name, then acting on them immediately, not for crafting complex metadata queries.

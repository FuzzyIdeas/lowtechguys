# rcmd - Press Kit

## About

rcmd is a reimagined app switcher for macOS. Hold the ⌘ Right Command key and press the first letter of an app's name to focus it instantly. App switching at the speed of thought.

The same key combo launches apps that aren't running, fuzzy-searches apps and windows, and switches Spaces by number with no animation. The Left Command key stays untouched, so the default macOS shortcuts keep working.

## Features

- Instant app switching: hold ⌘ Right Command, press the first letter of an app to focus it
- Launches apps that aren't running with the same key combo
- Dynamic key assignments out of the box, plus custom letter assignments
- Fuzzy search across apps and open windows, just hold, type, release
- Command-Tab replacement that filters out minimized windows and windows on other Spaces
- Instant Space switching by number, with no sliding animation
- Move the focused window to another Space and follow it there
- Stages: save and restore window workspaces
- Window jumping with ⌥ + letter, same-app cycling with Cmd-backtick
- On-screen key hints through the companion Keylume keyboard
- 16 built-in themes and extensive customization
- Mouse follows the focused app

## Use Cases

- Jumping straight to any app without cycling through Command-Tab
- Finding a specific window across Spaces by typing its name
- Rearranging windows across Spaces entirely from the keyboard
- Saving and restoring window layouts for different tasks with Stages
